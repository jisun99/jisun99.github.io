Lane Detection Domain Adaptation Sim-to-Real

Context-aware Sim-to-real Unsupervised Domain Adaptation for Lane Detection via Disentangled Feature Alignment

IEEE Transactions on Automation Science and Engineering 2025

This paper addresses the domain gap between simulation and real-world data in lane detection. By introducing context-aware disentangled feature alignment, the work aims to improve unsupervised domain adaptation performance and make lane detection models more effective in practical driving environments.

Citation

Paper Overview

The main motivation of this paper is that models trained in simulation often struggle to maintain performance when transferred to real-world driving data. In lane detection, this sim-to-real gap can make practical deployment difficult, even when synthetic data is abundant and convenient for training.

To address this challenge, the paper explores unsupervised domain adaptation with a focus on feature disentanglement and context-aware alignment. The goal is to better preserve task-relevant lane information while reducing the mismatch between synthetic and real domains.

Method Summary

  • Addresses unsupervised domain adaptation for lane detection under sim-to-real conditions
  • Introduces context-aware feature alignment to better capture lane-related information
  • Uses disentangled representations to separate transferable and domain-specific features
  • Aims to improve generalization from synthetic training data to real driving scenarios

Key Contributions

  • Presented a context-aware approach to sim-to-real unsupervised domain adaptation for lane detection
  • Proposed disentangled feature alignment for more effective domain transfer
  • Improved understanding of how contextual information can support robust lane perception
  • Contributed to practical autonomous driving perception research

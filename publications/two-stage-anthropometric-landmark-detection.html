Two-stage Anthropometric Landmark Detection Framework on Human Point Cloud: Dynamic-invariant and Static-specific Strategy

Engineering Applications of Artificial Intelligence Accepted · 2026

This paper presents a two-stage framework for anthropometric landmark detection on 3D human point clouds. The proposed method is designed to handle dynamic posture variation while also improving point-level precision through localized refinement on the input data.

Journal Page Citation

Paper Overview

The main goal of this paper is to improve landmark detection on 3D human point clouds under posture variation. Existing approaches often struggle either with dynamic body deformation or with obtaining precise landmarks directly on the original input surface. To address these limitations, the paper introduces a two-stage strategy that first estimates posture-invariant landmarks and then refines them on the input point cloud.

This work connects robustness and precision within a unified framework, enabling more reliable anthropometric landmark detection across both static and dynamic postures.

Method Summary

  • Stage 1 estimates pose-invariant anthropometric landmarks using a parametric human template representation
  • Stage 2 refines landmark positions directly on the input point cloud for improved point-level accuracy
  • The framework combines robustness to posture variation with localized geometric refinement
  • Experiments validate the effectiveness of the approach on 3D human scan datasets

Key Contributions

  • Proposed a two-stage framework for 3D anthropometric landmark detection
  • Addressed landmark estimation under dynamic posture variation
  • Improved point-level precision through localized refinement on the input point cloud
  • Demonstrated the applicability of deep learning-based landmark detection to practical 3D human analysis

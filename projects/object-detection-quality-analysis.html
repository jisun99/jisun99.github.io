Standardization of Ship Painting Quality Using Object Detection

Industrial Vision AI Project 2025.01 – 2025.12

This project focused on applying object detection-based AI to ship painting defect inspection in order to make quality evaluation more objective and efficient. By addressing the limitations of inspector-dependent judgment and high labor requirements, the project aimed to establish a more standardized inspection framework and ultimately contribute toward owner no-inspection.

Project Overview

The goal of this project was to apply object detection models to ship painting defect inspection and build a more objective quality assessment process through AI. In conventional ship painting inspection, defect evaluation often depended heavily on the subjective judgment of inspectors, which made it difficult to maintain consistent standards and also required significant manpower.

To address these limitations, this project introduced AI-based defect detection as a practical approach for standardizing inspection criteria and improving efficiency. Beyond model development itself, the project was also meaningful in that it pursued a broader industrial objective: supporting a more reliable inspection process and ultimately contributing to the achievement of owner no-inspection.

What I Worked On

  • Applied object detection-based AI to ship painting defect inspection tasks
  • Worked on building a more objective and standardized inspection process
  • Analyzed defect inspection tasks that had previously depended on inspector judgment
  • Improved AI-based inspection performance with consideration for practical field usage
  • Contributed to the development of a system direction aimed at reducing labor dependency and enhancing inspection consistency

Key Focus

  • Ship painting defect detection using object detection models
  • Reduction of subjectivity in conventional inspection processes
  • Improvement of inspection efficiency and reduction of manpower burden
  • Standardization of quality evaluation criteria through AI
  • Long-term contribution toward owner no-inspection

Outcome

This project helped me build a stronger understanding of how Vision AI can be used not only to improve detection performance but also to transform industrial inspection processes themselves. It expanded my perspective on AI as a tool for standardization, efficiency improvement, and practical value creation in real shipbuilding environments.

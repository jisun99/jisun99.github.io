Anthropometric Data Data Management System

Integrated Human Anthropometric Data Management System

National Research Project 2024.06 – 2024.12

This project focused on building an integrated system for managing human anthropometric data more efficiently. It provided an opportunity to understand how measurement-related data could be organized, processed, and utilized in a more systematic way for practical applications.

Project Overview

The main purpose of this project was to develop a system that could support integrated management of anthropometric data. The work involved thinking about how human body measurement data should be stored, organized, and made usable for efficient analysis and application.

Through this project, I expanded my understanding of human-centered data processing and learned how structured data management can support downstream analysis, research, and practical use in related systems.

What I Worked On

  • Participated in the development of an integrated anthropometric data management system
  • Worked on data organization and processing for more efficient utilization
  • Built experience in handling measurement-related human data
  • Improved understanding of system-oriented approaches to human-centered data management

Outcome

This project strengthened my ability to think not only about model development, but also about how data should be managed and utilized in practical systems. It broadened my perspective on the relationship between structured data management and applied AI or analytics workflows.

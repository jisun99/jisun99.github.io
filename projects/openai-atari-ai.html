Personal / Academic Project 2022.03 – 2022.06 Deep Learning Reinforcement Learning

OpenAI-based Atari Game Playing AI Implementation

Undergraduate Project

This project explored how AI can interact with dynamic environments through game playing. I implemented a deep learning-based Atari game playing AI and focused on improving interaction performance through action refinement and image processing strategies.

Back to Projects GitHub

Project Overview

The goal of this project was to build an AI agent capable of playing Atari games and to understand how learning-based agents can improve performance in interactive environments. Through this project, I gained practical experience in AI implementation, model behavior analysis, and iterative refinement of decision-making strategies.

A key focus was not only making the agent learn, but also improving the quality of interaction by refining how actions were selected and how visual inputs were processed. This project became an important starting point for my interest in AI and later Vision AI research.

What I Worked On

  • Implemented an Atari game playing AI based on deep learning methods
  • Refined the action strategy to reduce unnecessary actions and improve agent behavior
  • Applied image processing techniques to support more effective game-state understanding
  • Analyzed model performance and interaction quality in a dynamic game environment

Outcome

This project helped me build my first hands-on understanding of AI development in an interactive environment. It also became the foundation for my continued interest in computer vision, deep learning, and practical AI systems that can perceive and respond to changing conditions.

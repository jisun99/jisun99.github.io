Development of an AI-based Agent for Ship Painting Quality Inspection

Industrial Vision AI Project 2026.01 – Present

This project focuses on developing an AI-based agent for ship painting quality inspection. It combines object detection-based defect inspection across diverse ship hull blocks with SLAM-based localization for defect position estimation, while also building the connected database, server, and application infrastructure required for practical in-house use.

Project Overview

This project is an advanced in-house Vision AI initiative for ship painting quality inspection. Its main objective is not only to detect coating defects from images, but also to support more practical inspection workflows by handling diverse ship hull blocks and estimating the spatial location of detected defects.

To achieve this, the project combines object detection-based defect recognition with SLAM-based localization. In addition, it includes the development of the supporting database, server, and program infrastructure connected to an in-house worker app, enabling the system to function as a more integrated inspection platform rather than as a standalone AI model.

What I Worked On

  • Developed object detection-based models for defect inspection in ship painting environments
  • Worked on defect detection across diverse ship hull blocks under practical field conditions
  • Integrated SLAM to estimate and manage the spatial location of detected defects
  • Built connected system components including database, server, and application-linked programs for in-house worker usage
  • Contributed to shaping the project as an integrated inspection agent rather than a model-only solution

Key Focus

  • Inspection support across multiple ship hull block environments
  • SLAM-based defect localization and position estimation
  • Integration of AI models with database, server, and worker-facing applications
  • Practical system design for real industrial deployment
